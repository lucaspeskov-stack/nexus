# Atelier da Beleza

Site premium para o salão de beleza Atelier da Beleza, localizado no Centro do Rio de Janeiro.

## 🚀 Tecnologias

- React 18
- Vite
- Tailwind CSS
- Lucide React (ícones)

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 📱 Seções

- Hero com CTA
- Serviços oferecidos
- Depoimentos de clientes
- Sobre o negócio
- Localização com Google Maps
- Rodapé com contato

## 🎨 Características

- Design premium e moderno
- Totalmente responsivo
- Animações suaves no scroll
- Interativo com efeitos hover
- Otimizado para conversão
