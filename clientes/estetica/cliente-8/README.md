# I AM Salon & Spa — Site

Site institucional premium do **I AM Salon & Spa**, salão e spa em Ipanema, Rio de Janeiro.
Cabelo, estética, spa e bem-estar. Construído em **React + Vite + Tailwind CSS** com
animações em **Framer Motion**.

## ✨ Recursos

- **Hero** elegante com foto em destaque (efeito Ken Burns) e prova social
- **Serviços** com 6 especialidades em cards interativos (cabelo, estética, spa, unhas, cílios, maquiagem)
- **Galeria** de ambiente e trabalhos em grid responsivo
- **Avaliações** do Google (nota 4,8 · 137 avaliações) com distribuição e tags
- Seção **Sobre** com destaques e números de confiança
- **Mapa do Google** embutido + horários com indicador "aberto agora" em tempo real
- **Contato** com formulário que monta a mensagem direto no WhatsApp
- Botão flutuante e CTAs de **WhatsApp** em todo o site
- Responsivo, acessível (WCAG AA), com suporte a `prefers-reduced-motion`

## 🎨 Identidade

- **Paleta:** ameixa profunda + rose gold + malva + cremes
- **Tipografia:** Playfair Display (títulos) + Inter (texto)

## 🚀 Como rodar

Pré-requisito: Node.js 18+.

```bash
npm install
npm run dev      # ambiente de desenvolvimento
npm run build    # build de produção (pasta dist/)
npm run preview  # pré-visualiza o build
```

## 📝 Editar conteúdo

Todo o conteúdo (textos, contatos, serviços, avaliações, horários e imagens) fica
centralizado em [`src/data/content.js`](src/data/content.js).
